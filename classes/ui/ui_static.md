#### Класс CUIStatic

Базовый класс, представляющий статический элемента управления.

<br/>

##### Наследуется от:

CUIWindow

<br/>

##### Свойства

Данный класс не имеет свойств 

<br/>

| Сигнатура                                                    | Возвращаемое значение | Описание                                       |
| ------------------------------------------------------------ | --------------------- | ---------------------------------------------- |

| SetTextX(number *posX*)                                      | Нет                   | Установка позиции текста по оси X              |
